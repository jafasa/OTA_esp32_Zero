# OTA_esp32_Zero

این مخزن شامل نمونه کدها و پروژه‌هایی برای پیاده‌سازی بروزرسانی نرم‌افزار (OTA - Over The Air) روی برد ESP32 به صورت ساده و عملی است.

## معرفی

بروزرسانی OTA قابلیتی است که به شما امکان می‌دهد برنامه اجرا شده روی برد ESP32 را بدون نیاز به اتصال فیزیکی، از راه دور و از طریق اینترنت یا شبکه وای‌فای بروزرسانی کنید. این پروژه با هدف آموزش و ارائه یک پیاده‌سازی ساده برای کاربران تازه‌کار و حرفه‌ای ساخته شده است.

## ویژگی‌ها

- پیاده‌سازی بروزرسانی OTA برای ESP32
- کدنویسی ساده و قابل فهم
- امکان توسعه و شخصی سازی
- توضیح قدم به قدم راه‌اندازی

## راه‌اندازی

۱. کد پروژه را به برد ESP32 خود آپلود کنید.
۲. پارامترهای وای‌فای و سرور OTA را مطابق نیاز تغییر دهید.
۳. پس از آپلود اولیه، بروزرسانی نرم‌افزاری به صورت OTA ممکن خواهد بود.

## نیازمندی‌ها

- برد ESP32
- آردوینو IDE یا PlatformIO
- کتابخانه‌های مورد نیاز: `WiFi.h`, `Update.h` و سایر وابستگی‌ها

## نحوه انجام OTA

برای بروزرسانی برنامه، کافیست فایل باینری جدید را از طریق پنل OTA یا نرم‌افزار مربوطه به برد ارسال کرده و روند بروزرسانی را طی کنید. پس از تکمیل موفقیت‌آمیز، ESP32 ری‌استارت شده و برنامه جدید اجرا خواهد شد.
